# TwiTrends

University course project focused on software development concepts.

## What I worked on
- Frontend dashboard (UI)
- Backend API routes & middleware
- Basic database configuration and integration

## Modules
- `my-dashboard`: frontend
- `Backend`: backend API
- `twi-scraper`: data collection scripts
- `twi-report`: reporting script

## Notes
Sensitive information (passwords, tokens, cookies) is removed and handled via environment variables.
